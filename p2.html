Lispの基本文法

Lispはリスト構造を基本とする関数型プログラミング言語です。

基本構文

  • すべての式は括弧で囲む: (+ 1 2)
  • 演算子が最初に来る前置記法
  • リストはプログラムの基本単位

変数定義

(setq x 10)  ; xに10を代入

関数定義

(defun square (x)
  (* x x))  ; xの二乗を計算

条件分岐

(if (> x 0)
    (print "Positive")
    (print "Non-positive"))

ループ

(dotimes (i 5)
  (print i))  ; 0から4を出力
