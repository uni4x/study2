Lispのdefunの書き方

defun はLispで関数を定義するための構文です。

基本構文

(defun 関数名 (引数1 引数2 ...)
  関数の本体)

例:

(defun add (x y)
  (+ x y))

この関数は (add 2 3) のように呼び出すと 5 を返します。

デフォルト引数なしの関数

(defun greet ()
  (print "Hello, Lisp!"))

この関数は引数を取らずに呼び出すと「Hello, Lisp!」と表示します。

条件分岐を含む関数

(defun check-positive (n)
  (if (> n 0)
      "Positive"
      "Non-positive"))

この関数は、引数 n が正の数かどうかを判定します。

再帰関数

(defun factorial (n)
  (if (<= n 1)
      1
      (* n (factorial (- n 1)))))

この関数は n の階乗を計算します。
